# Solitude

Solitude is a web project built using **React**, **Vite**, **Tailwind CSS**, and **ESLint**. It provides a fast development environment and a clean project setup.

## Tech Stack

- ⚛️ React 19
- ⚡ Vite 6 (with HMR)
- 🎨 Tailwind CSS
- ✅ ESLint

## Getting Started

To run the project locally:

```bash
npm install
npm run dev
